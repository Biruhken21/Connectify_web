# Connectify Web - Social Networking Platform

A full-stack social networking web application built with React (frontend) and Node.js/Express (backend) to connect Startups with Investors, share posts, and build communities.

## üöÄ Features

### Frontend (React)
- **Modern UI/UX**: Responsive design with clean interface
- **User Authentication**: Secure login and registration
- **Social Feed**: Browse and interact with posts
- **User Profiles**: Customizable profiles with bio and avatar
- **Real-time Updates**: Live notifications and updates
- **Friends System**: Add friends and manage connections
- **Media Sharing**: Upload and share images/videos

### Backend (Node.js/Express)
- **RESTful API**: Well-structured API endpoints
- **JWT Authentication**: Secure token-based authentication
- **MongoDB Database**: Scalable NoSQL database
- **File Upload**: Image and media upload support
- **Real-time Features**: Socket.io for live updates
- **User Management**: Complete user CRUD operations
- **Post Management**: Create, read, update, delete posts
- **Security**: Input validation and sanitization
